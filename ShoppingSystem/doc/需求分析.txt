程序分类：
.idea
bin 所有可执行文件，所有代码的入口
config 所有的配置文件，
db 存储的相关信息
lib 所有通用的代码，公共的类库
src 所有的业务代码


db- admin 保存所有的管理员，每个管理员就是一个文件
db- user  每个用户就是一个文件,用卡号做文件夹名，每一个人有自己的记录和用户的基本信息（卡号，用户名，密码，信用卡额度，本月剩余额度，储蓄账户金额，注册时间，到期时间，状态（normal,locked,disable）,欠款记录）
